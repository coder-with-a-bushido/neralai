A simple <img align="center" height="20" src="./static/neralai.png"> client to demo streaming your camera and viewing that stream in a browser.

![Neralai Client Demo](./static/neralai_demo.gif)

Install dependencies

```sh
npm i
```

Run the demo

```sh
npm run dev -- --open
```
